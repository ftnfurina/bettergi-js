# BetterGI JS

BetterGI JS 脚本辅助库，可以帮助你快速构建和调试 BetterGI JS 脚本。

## 🌟 特性

- 基于 TypeScript，提供 BetterGI JS 脚本开发所需的类型定义。
- 基于 Rollup 支持动态打包，搭配 BetterGI 即可实时调试脚本。
- 支持 `manifest.json` 和 `settings.json` 配置文件 schema 验证。

## ⚙️ 使用

```bash
npm create bettergi-js
```
